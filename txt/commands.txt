socketstart - Start web socket server at standard port: 3002
socketstart [port number] - Start web socket server at specified port
socketstop - Stop web socket server
webstart - Start web server at standard port: 8080
webstart [port number] - Start web server at specified port
webstop - Stop web server
auto - starts both servers on their standard ports(simple setup)
max [number] - Set maximum number of connections
save - Save all text in the log window to file
clear - Clear the log window
help - Display this message
exit - Close web socket server, save log file and exit program
